# epd2in13b_v4
# ZJYE213S08R0G01 
2.13 inches e-paper display. Screen resolution 122x250 pix


# Pinout
| e-Paper            | Raspberry Pi | ESP32 (30 pins) |
|--------------------|--------------|-----------------|
|  3.3v              |  1           | 3.3v |
|  GND               |  20          | GND  |
|  SCL (CLK or SCK)  |  23          | 18   |
|  SDA (DIN or MOSI) |  19          | 23   |
|  RES (RST)         |  11          | 16   |
|   DC               |  22          | 17   |
|   CS               |  24          | 5    |
|  Busy              |  18          | 4    |
